# Changelog

## 0.2.0

- Refactored the project into an installable `src/` package.
- Closed URI-policy bypasses in plain-text mode.
- Added canonical HTTPS parsing, IDNA handling, Unicode-format rejection, credential blocking, port validation, and optional DNS checks.
- Added redacted Wi-Fi previews, hidden password input, stdin/file password support, UTF-8 byte checks, and 64-character PSK support.
- Added atomic no-clobber output, random filenames, secure permissions, color contrast enforcement, and decode-after-render verification.
- Added passive decoding and risk analysis.
- Added Ed25519 signed envelopes and AES-256-GCM encrypted envelopes.
- Added CSV/JSON batch generation, SVG output, geographic, calendar, and JSON payloads.
- Added automated tests, CodeQL, dependency auditing, Dependabot, and project governance files.
